# Stock Trading Platform

## 📌 Project Description
The Stock Trading Platform is a Java console-based application developed as part of the **CodeAlpha Java Programming Internship**. The application simulates a basic stock trading environment where users can view available stocks, buy and sell shares, and track their portfolio performance.

## 🚀 Features
- 📈 Display available stocks and market prices
- 💰 Buy stocks
- 📉 Sell stocks
- 📊 Track portfolio performance
- 💼 Display cash balance and total portfolio value
- 🖥️ Console-based user interface
- 🧩 Object-Oriented Programming (OOP) implementation

## 🛠️ Technologies Used
- Java
- Object-Oriented Programming (OOP)
- HashMap
- Collections Framework
- Scanner Class

## ▶️ How to Run

1. Clone this repository:
   ```
   git clone https://github.com/YourUsername/CodeAlpha_StockTradingPlatform.git
   ```

2. Compile the program:
   ```
   javac StockTradingPlatform.java
   ```

3. Run the program:
   ```
   java StockTradingPlatform
   ```
